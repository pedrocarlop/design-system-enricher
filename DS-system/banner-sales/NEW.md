# banner-sales

## What this component is

A promotional banner at the top of the page announcing NN/G self-paced training.

## Where it is used

At the very top of the page, above the header and main navigation.

## How it is used

It presents a short promotion message and a CTA link, with a dismiss button on the right.

## Structure

`div.banner-sales` with `div.banner-sales__content` and a sibling `button`; the content block contains `p.banner-sales__text` and `a.banner-sales__action`.

## Variants

None explicitly defined

## Layout patterns

Standalone full-width strip.

## Relationships with other components

Appears before `nav-main` inside the page chrome.

## Usage rules (inferred)

Used once per page to communicate a temporary promotion before the user enters the main content.

## Content patterns

Short promo copy, a single CTA, and a close control.
