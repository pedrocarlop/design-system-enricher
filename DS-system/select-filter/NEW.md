# select-filter

## What this component is

A single dropdown filter in the self-paced catalog.

## Where it is used

Inside `selfpaced-search`, in the filter header above the course list.

## How it is used

It lets users narrow the course list by topic, difficulty, price, or duration.

## Structure

`div.select-filter.default-filter` with a `label.label-large` and a native `select`.

## Variants

`default-filter`

## Layout patterns

Repeated filter control in a horizontal filter row.

## Relationships with other components

Grouped with three other `select-filter` controls.

## Usage rules (inferred)

Used four times to support browsing and sorting within the catalog.

## Content patterns

Label text plus select options.
