# accordion-item

## What this component is

A single FAQ accordion row.

## Where it is used

Inside the `faqs` accordion.

## How it is used

It expands and collapses one question and answer pair.

## Structure

`details.accordion-item` with `summary.label-xl-600`, `div.divider`, and `div.content`.

## Variants

None explicitly defined

## Layout patterns

Vertical stack of repeated disclosure items.

## Relationships with other components

Always nested inside `faqs`.

## Usage rules (inferred)

Used ten times to cover the page's FAQ set.

## Content patterns

Question text in the summary and answer content in the body.
