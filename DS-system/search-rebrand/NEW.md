# search-rebrand

## What this component is

The site search form used in the top navigation.

## Where it is used

Inside `nav-main`, on the right side of the header.

## How it is used

It gives visitors a search entry point with inline form controls.

## Structure

`form.search-rebrand[role="search"]` with `div.search-rebrand__field`, a text input, a reset button, and a submit button.

## Variants

None explicitly defined

## Layout patterns

Inline header control.

## Relationships with other components

Grouped with `cart-and-profile` inside `search-and-profile`.

## Usage rules (inferred)

Used once in the header to support site search from any page state.

## Content patterns

Search text, field controls, and form actions.
