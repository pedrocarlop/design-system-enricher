# nav-main

## What this component is

The primary desktop navigation for the NN/G site.

## Where it is used

Inside the page header.

## How it is used

It provides the top-level site sections, submenu navigation, search, cart status, and login access.

## Structure

`nav.nav-main.nav-main--rebrand.desktop` containing `div.ds-wrapper.header-content`, which splits into `div.nav-menu` and `div.search-and-profile`.

## Variants

`nav-main--rebrand`, `desktop`

## Layout patterns

Horizontal header navigation with grouped submenu lists.

## Relationships with other components

Appears before `catalog-banner-header` and after the top promo banner.

## Usage rules (inferred)

Used once as the site-wide navigation shell on this page.

## Content patterns

Burger button, logo, four top-level nav groups, search, cart, and login.
