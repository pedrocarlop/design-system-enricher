# cart-and-profile

## What this component is

The header utility area for cart status and sign-in access.

## Where it is used

Inside `nav-main`, to the right of the search form.

## How it is used

It shows the cart as empty and provides a login link.

## Structure

`div.cart-and-profile` with `div.cart.empty` and `div.login`.

## Variants

`empty`

## Layout patterns

Inline header utility group.

## Relationships with other components

Follows `search-rebrand` within `search-and-profile`.

## Usage rules (inferred)

Used once as a compact header status area.

## Content patterns

Cart state text and login entry point.
