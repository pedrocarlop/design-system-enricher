# links-list

## What this component is
The footer navigation list of grouped links.

## Where it is used
- Inside `footer .navigation`.

## How it is used
- It presents the site's footer navigation in four grouped columns.

## Structure
- Root element: `div.links-list`
- Contains a top-level `ul`
- That `ul` contains 4 top-level `li` groups
- Each column item contains a heading and a nested `ul`

## Variants
None explicitly defined

## Layout patterns
- Multi-column grouped list.

## Relationships with other components
- Part of the `footer-rebrand` shell.
- Sits between `newsletter` and `bottom-footer`.

## Usage rules (inferred)
- Used once to expose the main site navigation categories at the bottom of the page.

## Content patterns
- Column headings.
- Stacked text links.
