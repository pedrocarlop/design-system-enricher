# back-to-top

## What this component is

The floating button that returns the user to the top of the page.

## Where it is used

After the footer in the body.

## How it is used

It gives users a quick way to jump back to the top after scrolling the page.

## Structure

`button.back-to-top.ds-button-medium.ds-button-gray.label-m-500`

## Variants

`ds-button-medium`, `ds-button-gray`

## Layout patterns

Single floating action button.

## Relationships with other components
